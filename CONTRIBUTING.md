# Contributing to dtZahlensysteme

Thank you for considering contributing! Your help makes this number converter better for everyone.

## How to Contribute

### Reporting Bugs
- Check if the issue already exists
- Open a new issue with:
  - Clear title describing the bug
  - Steps to reproduce
  - Expected vs actual behavior
  - Browser/PHP version info

### Suggesting Features
- Open an issue with the "enhancement" label
- Describe the feature and its use case
- Be specific about expected behavior

### Code Contributions

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/dtZahlensysteme.git`
3. **Create branch**: `git checkout -b feature/your-feature`
4. **Make changes** following our style guide
5. **Test** your changes thoroughly
6. **Commit**: `git commit -m "Add: description of change"`
7. **Push**: `git push origin feature/your-feature`
8. **Open Pull Request** with detailed description

### Code Style

**PHP:**
- Follow PSR-12 coding standards
- Use meaningful variable names in English
- Add PHPDoc comments for functions
- Use type hints where possible

**JavaScript:**
- Use ES6+ features
- Follow Airbnb style guide
- Comment complex algorithms

**General:**
- Keep functions under 50 lines
- Write unit tests for new features
- Update README if adding features
- Add German and English translations

### Testing

```bash
# Run PHP tests
composer test

# Manual testing checklist
- [ ] Test all number system conversions
- [ ] Check mobile responsiveness
- [ ] Verify database operations
- [ ] Test in German and English
```

## Questions?

Open an issue or start a discussion. We're happy to help!

Thank you for making dtZahlensysteme better!
